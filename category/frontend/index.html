vue

vue 实现了变量与模型的双向绑定

减少 dom 操作（虚拟 dom），提高渲染效率，双向数据绑定

特点及优势

  • 利用虚拟 DOM 渲染
  • 轻量级
  • 响应式组件
  • 支持服务器渲染
  • 易于集成路由工具，打包工具以及状态管理工具

条件判断

v-if ：变量值为 true/false，支持表达式 大于小于等于，如果为 false，直接没有这条数据
v-show ：变量值为 true/false，支持表达式 大于小于等于，如果为 false，实际是隐藏数据
反复使用 用 v-show，减少渲染性能更好，只使用一次 用 v-if
